# tcp-port-scanner
A simple Python script for scanning ports on a given host.

## Features
- Scans a range of ports on a specified host.
- Identifies common services associated with open ports.

## Common Ports Identified: 

    21: FTP
    22: SSH
    23: Telnet
    25: SMTP
    53: DNS
    80: HTTP
    110: POP3
    135: RPC
    139: NetBIOS
    143: IMAP
    443: HTTPS
    445: SMB
    1433: MSSQL
    3306: MySQL
    3389: RDP
    5900: VNC
